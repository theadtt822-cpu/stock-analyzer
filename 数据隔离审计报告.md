# StockHolmes 数据隔离审计报告

## ✅ 已正确隔离

### 1. dashboard_server.py (交互式仪表盘, 端口8082)
- `USERS` 字典：天天→`output/boss/`, 波波→`output/boyfriend/`
- 所有路由检查 `user_key not in USERS`
- `generate_report_html()` → 只保存到请求者的目录（**刚修复**）
- `check_report_exists()` → 只检查请求者的目录（**刚修复**）
- `load_json/save_json` → 每个用户独立的 portfolio/watchlist 文件
- 报告展示页：`/{user_key}/reports/{filename}` → 只从用户自己的目录读取

### 2. report_server.py (旧报告中心, 端口8081)
- 两个独立密钥路径：`tiantian_reports_8k3m`→`output/boss/`, `bobo_reports_9x7n`→`output/boyfriend/`
- 路径穿越防护：`os.path.realpath` 校验
- 根路径返回403，不暴露用户列表

### 3. gen_dashboard_v2.py (持仓仪表盘生成)
- `python gen_dashboard_v2.py boss|boyfriend|all`
- boss → 读 `output/boss/portfolio_dashboard.json`，写 `output/boss/`
- boyfriend → 读 `output/portfolio_data.json`，写 `output/boyfriend/`

### 4. watchlist_analyze.py (自选股分析)
- `--user boss|boyfriend` 参数
- boss → 输出到 `output/boss/watchlist/`
- boyfriend → 输出到 `output/boyfriend/watchlist/`

### 5. web_analyzer.py (网页版分析工具)
- 前端选择用户后，报告正确复制到对应用户目录
- 报告读取从用户专属路径 `output/{user}/`
